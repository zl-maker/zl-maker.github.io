自学JAVA学习路线及资源推荐

发表于2022-02-26|更新于2022-04-25|技术
|字数总计:1.6k|阅读时长:4分钟|阅读量:

提要

面试的话，一般原理占 80%，实践占 20%，所以当要准备面试的时候，尽量突击原理部分，当然平时也需要练习大量的 java 代码，跟着视频一步一步得敲，不要嫌慢，应当注重长远发展。但本篇主要针对那些还有一两年才会面试的同学，因此只会列一个大纲，不会很详细的将面试题列出来，仅仅充当学习的路径。可以当作 java 学习路径，以我自己的经历来写，不代表绝对，避免同学们走弯路，学习不必要的东西或者过时的，是专门针对面试写的。

另外本篇只提供视频资料，书籍的话，我认为作为小白不是很推荐，当后面工作的时候或者有闲余时间时，再读书籍，来升华自己，提升职位。

学习顺序也可以按照本篇顺序来学习。

本篇是针对至少学过 c 语言的小伙伴的，不然会学得很艰难。

java 基础

这部分我认为是你学 java 最重要的部分，毕竟是基石，后面所学的框架什么，都要建立这个基础之上，因此要重点学习，认真听讲，老师所写的代码和注释都要跟着敲一遍，不要偷懒！！！这样才记忆深刻，慢一点没关系，不要受到其他人的影响！！！

推荐视频

尚硅谷 Java 入门视频教程(在线答疑+Java 面试真题)

推荐原因

真心推荐，讲的很细、很有趣，也会分享面试的经验。


MySQL 基础篇

后端开发的基本能力之一，非常重要，要跟着视频多敲，多刷题。

推荐视频

老杜带你学_mysql 入门基础（mysql 基础视频+数据库实战)

推荐原因

对 mysql 基础操作练习是比较全的，可以打下很好的基础。


计算机基础

没有计算机基础，后面的课程会上的很懵的。如果是科班学生，一定要好好学习，争取拿满分。

数据结构与算法

考量程序员上限的方面，几乎是你一生要追求的，面试必考必问。

推荐视频

数据结构与算法（Java 数据结构与算法）

推荐原因

用 java 写数据结构和算法，巩固了 java 基础的练习


操作系统

重点学习进程管理、存储管理这两个方面。

推荐视频

操作系统

推荐原因

推荐视频

计算机网络

推荐原因

以动画来展示，比较有趣、直观。


MySQL 高级篇
